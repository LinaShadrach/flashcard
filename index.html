instructions

  • JavaScript(JS)A programming language used to make web pages interactive.
  • OperatorsAn operator is a special character (or characters) that indicates an action to be performed.
  • VariablesContainers used to store information; allow for a way to label data with a descriptive name.
  • Variable naming conventionsRules for assigning variables (in Javascript): Should begin with a letter, case sensitive, clear & descriptive names, named in a way easy for other developers to understand.
  • FunctionA block of code that performs an action and returns a result; optionally takes arguments
  • MethodsActions run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.
  • ArgumentsAdditional information given to some methods to help them perform their actions.
  • ParametersVariables that are assigned to arguments.
  • ReturnValue of the method's response.
  • Chaining methodsAct of calling a method directly on the return value of another method.
  • StringsA JavaScript data type that represents the exact text of whatever is enclosed in the quotes.
  • BooleansA data type that can only be valued as true or false.
  • UndefinedAn object (such as a variable) without a defined value; also a data type.
  • NaNStands for not a number; a type of number.
  • EscapeTells JavaScript that the " that comes right after it is not the end of the string, but just a character inside the string; represented as //.
  • AlertA function that pops up a dialog box with the string that you pass in as an argument.
  • CommentsAnything following // on a line in JavaScript is ignored by the computer. Comments are a convenient way to leave notes in your code for yourself or other programmers.
  • jQueryA JavaScript library that makes it easier to incorporate JavaScript on your web pages.
  • AttributesProvide addtional information about HTML elements that Javascript can then manipulate.
